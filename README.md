# muelsyse.com

This is the frontend code for [muelsyse.com](https://www.muelsyse.com).

## Building and developing

Just like any other vue projects. Run 
```shell
npm install
```
to install all the dependencies.

To run the development server, just run 
```shell
npm run dev
```

Build build, run
```shell
npm run build
```
